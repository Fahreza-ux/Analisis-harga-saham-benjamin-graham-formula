
# Kalkulator Valuasi Saham Sederhana
Proyek ini adalah alat bantu sederhana berbasis web untuk menghitung estimasi **Harga Wajar Saham** menggunakan metode **Benjamin Graham**. Alat ini dirancang khusus untuk mempermudah investor agar tidak perlu berurusan dengan angka-angka besar (miliaran/triliunan) saat melakukan analisis cepat.
## 🚀 Fitur Utama
 * **Input Ringkas:** Menggunakan format *Per Share* (EPS) langsung dari aplikasi saham Anda.
 * **Rumus Graham:** Menggunakan formula klasik V = EPS \times (8,5 + 2g) untuk menilai potensi harga wajar.
 * **Ramah Pengguna:** Tidak perlu kalkulator eksternal, hasil langsung muncul secara otomatis.
 * **Tanpa Instalasi:** Cukup buka file .html di browser apa saja.
## 📋 Cara Penggunaan
 1. Buka file index.html di browser Anda (Chrome, Edge, Firefox, dll).
 2. Lihat data saham dari aplikasi sekuritas Anda (seperti MCOL atau IPCC).
 3. Masukkan angka **EPS (Earning Per Share)** ke kolom yang tersedia.
 4. Masukkan **estimasi pertumbuhan laba tahunan (%)** perusahaan tersebut (Contoh: gunakan 5% untuk konservatif).
 5. Klik tombol **"Hitung Harga Wajar"**.
 6. Harga wajar akan langsung muncul di bawah tombol.
## 💡 Konsep Rumus
Alat ini menggunakan **Rumus Benjamin Graham** yang populer di kalangan *value investor*:
 * **EPS (Earning Per Share):** Laba yang didapatkan perusahaan untuk setiap lembar saham.
 * **8,5:** Konstanta P/E rasio untuk perusahaan dengan pertumbuhan nol.
 * **2g:** Proyeksi pertumbuhan laba di masa depan.
> **Catatan Penting:** > Kalkulator ini hanya alat bantu hitung. Keputusan investasi tetap berada di tangan Anda. Selalu gunakan margin of safety saat melakukan pembelian saham.
