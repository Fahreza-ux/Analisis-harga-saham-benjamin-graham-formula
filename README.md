# Analisis-harga-saham-benjamin-graham-formula

Rumus yang Digunakan di Balik Layar
Agar analisis fundamental Anda semakin tajam, ini adalah formula matematis yang diaplikasikan pada kode di atas:
EPS (Earning Per Share)
EPS = \frac{Laba\ Bersih}{Jumlah\ Saham\ Beredar}
Mengukur berapa banyak keuntungan yang dihasilkan untuk setiap lembar saham.
BVPS (Book Value Per Share)
BVPS = \frac{Total\ Ekuitas}{Jumlah\ Saham\ Beredar}
Menunjukkan nilai aset bersih perusahaan per lembar saham jika perusahaan dilikuidasi hari ini.
PER (Price to Earning Ratio)
PER = \frac{Harga\ Saham\ Saat\ Ini}{EPS}
Menilai seberapa murah atau mahal harga saham dibandingkan dengan laba yang dihasilkannya.
PBV (Price to Book Value)
PBV = \frac{Harga\ Saham\ Saat\ Ini}{BVPS}
Mengevaluasi apakah pasar menghargai perusahaan di atas atau di bawah nilai buku aslinya.
Harga Wajar (Intrinsic Value - Benjamin Graham Formula)
V = EPS \times (8.5 + 2g)
Di mana V adalah Harga Wajar, 8.5 adalah P/E wajar perusahaan stagnan, dan g adalah persentase proyeksi pertumbuhan. Ini adalah cara klasik untuk mengetahui apakah harga saham saat ini sedang "salah harga" (undervalued) atau justru terlalu mahal (overvalued).
